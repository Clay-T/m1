Mission Statement

The project is to make a website for an ice cream shop to make it more user friendly and provide more services with the website. Customers will be able to online order and give feedback and there will be delivery services supported online.

Success Evaluation

We can evaluate its successfulness by the amount of orders that will come through the website.

Target Audience

The targeted audience are children, teenagers and young adults who enjoy having their ice cream at home. The user interface is made to cater and please the eyes and feel more inclined to order.

Content

  • Home Page - Introduction of the ice cream shop.
  • Products Page - Shows lists of ice cream the shop offers.
  • Upcoming Cafe Page - Provides the customers with the information about the upcoming cafe.
  • Events Page - Displays all events chronologically.
  • Services Page - Provides information on delivery details and party planners and exchanges and returns.
  • Order Page - Made for customers to easily use the website to order online.
  • Feedback Page - Made for customers to easily use the website to order online.
  • Contacts Page - Contact information.
  • Ogranisation Info Page - Information regarding to the organization.
  • FAQ Page
